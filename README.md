## Python 环境配置

### 前置条件

- Python 3.12

### 创建虚拟环境

```bash
python3.12 -m venv .venv
```

### 激活虚拟环境

**macOS/Linux:**
```bash
source .venv/bin/activate
```

**Windows:**
```bash
.venv\Scripts\activate
```

### 安装依赖

```bash
pip install -r requirements.txt
```

### 添加新依赖

```bash
pip install <package-name>
pip freeze > requirements.txt
```
