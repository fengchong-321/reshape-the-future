## 本周练习任务

### 必做任务

1. **提交 PR**
```markdown
# PR 描述模板

## 类型
- [ ] Bug fix
- [ ] Feature
- [ ] Refactor

## 描述
简短描述改动

## 测试
- [ ] 新增测试
- [ ] 现有测试通过

## 检查清单
- [ ] 代码通过 lint
- [ ] 测试通过
- [ ] 文档已更新
```

2. **跟进 review**
```
# 回应 maintainer 的反馈
# 及时修改代码
```

```markdown
## 背景
## 过程
## 结果
## 感想
```
