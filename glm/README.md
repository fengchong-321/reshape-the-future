## 学习资源

### 必备工具

```bash
# Python 环境
python --version  # 3.11+

# 虚拟环境
python -m venv venv
source venv/bin/activate

# 核心库
pip install pytest requests allure-pytest
pip install pymysql redis
pip install playwright pytest-playwright
pip install Appium-Python-Client

# AI 测试相关
pip install openai anthropic
pip install chromadb faiss-cpu pinecone-client
pip install sentence-transformers
pip install langchain langchain-community
pip install ragas deepeval
```

### 推荐资源

- [OpenAI API 文档](https://platform.openai.com/docs)
- [LangChain 文档](https://python.langchain.com/)
- [RAGAS 评测框架](https://docs.ragas.io/)
- [DeepEval 文档](https://docs.confident-ai.com/)
