# 2026 字节跳动高级测试开发 · 20周脱产学习计划

## 学习计划概览

### 第一阶段：Python 编程基础（第1-3周）

| 周次 | 主题 | 核心内容 |
|------|------|---------|
| [第1周](week01-python基础.md) | Python 基础语法 | 变量、列表、字典、字符串、控制流、函数 |
| [第2周](week02-面向对象.md) | 面向对象编程 | 类与对象、继承、封装、异常处理、上下文管理器 |
| [第3周](week03-文件与模块.md) | 文件操作与模块化 | JSON/YAML/CSV 处理、模块与包、项目结构 |

### 第二阶段：自动化测试框架（第4-6周）

| 周次 | 主题 | 核心内容 |
|------|------|---------|
| [第4周](week04-pytest基础.md) | Pytest 基础 | 测试函数、断言、fixture、参数化、标记 |
| [第5周](week05-pytest进阶.md) | Pytest 进阶与 Allure | fixture 进阶、conftest、钩子函数、Allure 报告 |
| [第6周](week06-接口自动化.md) | 接口自动化测试 | Requests、Session 管理、接口封装、认证 |

### 第三阶段：数据与服务端测试（第7-9周）

| 周次 | 主题 | 核心内容 |
|------|------|---------|

| [第8周](week08-Git版本控制.md) | Git 版本控制 | 分支管理、远程仓库、冲突解决、Git Flow |
| [第9周](week09-Linux基础.md) | Linux 基础 | 文件操作、进程管理、日志分析、Shell 脚本 |

### 第四阶段：CI/CD 与容器化（第10-11周）

| 周次 | 主题 | 核心内容 |

| [第11周](week11-Docker容器化.md) | Docker 容器化测试 | Dockerfile、Docker Compose、testcontainers |

### 第五阶段：性能测试（第12-13周）

## 每周文档结构

每个周文档都包含以下内容：

1. **学什么** - 明确的学习主题和重要性标注
2. **知识点详解** - 直接讲解，包含代码示例，不推荐看视频
3. **学到什么程度** - 必须掌握和应该了解的清单
4. **练习内容** - 具体的练习任务
5. **检验标准** - 自测题和答案
