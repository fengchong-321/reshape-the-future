1. 装饰器原理？

4. 生成器 vs 迭代器？

2. 参数化怎么做？
3. 如何生成测试报告？

**接口测试：**
1. 如何处理接口依赖？
2. 如何做数据驱动？
3. 如何保证测试数据隔离？

**性能测试：**
1. 如何定位性能瓶颈？

**AI 测试：**
1. LLM 测试的挑战？
2. 如何测试 RAG 系统？

## 四、模拟面试

### 模拟1：技术面试

```
Q: 介绍一下你做过的自动化测试框架？

回答框架：
1. 背景：为什么需要这个框架
2. 架构：整体设计
3. 技术栈：用了什么技术
4. 亮点：有什么创新
5. 成果：解决了什么问题
```

### 模拟2：场景题

```
Q: 如何测试一个 AI 客服机器人？

回答要点：
1. 功能测试
   - 意图识别准确率
   - 回答相关性
   - 多轮对话

2. 性能测试
   - 响应时间
   - 并发能力

3. 安全测试
   - 有害内容过滤
   - 越狱攻击防护

4. 评估指标
   - 准确率
   - 用户满意度
```

### 模拟3：算法题

```
Q: 实现一个 LRU 缓存

要点：
1. 说出数据结构选择（HashMap + 双向链表）
2. 画出结构图
3. 说出时间复杂度
4. 编写代码
```
