# Week 01 - Python 基础

> **技术更新声明**：以下计划基于 2026-03-29 之前的主流技术栈。建议你在执行前自行检索最新动态，或告诉我确认是否有新工具出现。

---

## 本周目标

掌握 Python 基础语法，能够编写简单的数据处理脚本和测试函数，为后续 pytest 学习打下基础。

---

## 概要步骤

- **Day 1-2**：环境搭建与基础语法
  - 安装 Python 3.11+、配置开发环境（VS Code 或 PyCharm）
  - 学习变量、数据类型（int、float、str、bool）
  - 学习字符串操作（格式化、切片、常用方法）

- **Day 3-4**：数据结构与控制流
  - 学习列表（list）、字典（dict）、集合（set）、元组（tuple）
  - 学习条件语句（if-elif-else）
  - 学习循环语句（for、while）

- **Day 5-6**：函数与模块
  - 学习函数定义、参数、返回值
  - 学习作用域、lambda 表达式
  - 学习模块导入、标准库使用（os、json、datetime）

- **Day 7**：综合练习与周检验
  - 完成综合练习题
  - 进行本周自测

12. 变量作用域（局部变量、全局变量）
13. Lambda 表达式
14. 模块导入与标准库
15. 文件读写基础
